PREET STUDIO WEBSITE - SETUP GUIDE

1. सबसे पहले index.html और script.js में 919999999999 की जगह अपना WhatsApp नंबर डालें।
2. index.html में फोन नंबर, ईमेल, Instagram और address बदलें।
3. अपनी photos लगाने के लिए:
   - एक "images" folder बनाएं।
   - photos के नाम wedding.jpg, couple.jpg, prewedding.jpg, bridal.jpg, album.jpg, cinematic.jpg रखें।
   - styles.css में .g1 से .g6 तक background बदलें, उदाहरण:
     .g1{background-image:url('images/wedding.jpg');}

FREE WEBSITE ONLINE कैसे करें:

OPTION 1 - NETLIFY
1. netlify.com खोलें।
2. Free account बनाएं।
3. पूरे website folder को drag and drop करें।
4. आपकी website तुरंत online हो जाएगी।

OPTION 2 - GITHUB PAGES
1. github.com पर free account बनाएं।
2. New repository बनाएं।
3. सभी files upload करें।
4. Settings > Pages > Deploy from branch > main चुनें।
5. Free website link मिल जाएगा।

CUSTOM DOMAIN:
Website free चलेगी, लेकिन .com या .in domain आमतौर पर paid होता है।
